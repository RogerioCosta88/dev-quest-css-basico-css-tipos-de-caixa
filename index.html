Tipos de Caixas

Bloc Level

É um elemento que ocupa a largura total da página. Um exemplo de bloc level são os paragráfos, títulos, etc, que quebram a linha quando terminam. O próximo elemento começará em outra linha.

Inline Level

Já o inline são elementos que não quebram a linha. Podemos modificar um elemento block para inline, os paragráfos abaixo são um exemplo de como ficará.
